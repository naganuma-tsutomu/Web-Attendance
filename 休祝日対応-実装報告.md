# 休祝日対応 実装報告

## 概要
シフト表アプリに日本の祝日対応を追加した。シフト生成時に祝日を考慮し、カレンダー上で祝日を視覚的に表示できるようにした。

## 実装内容

### 1. 祝日パッケージの導入
- `@holiday-jp/holiday_jp` パッケージを追加
- 内閣府公式データに基づく日本の祝日データを使用

```bash
npm install @holiday-jp/holiday_jp --save
```

### 2. データベーステーブル作成
`files/holidays_migration.sql` を作成：

```sql
CREATE TABLE IF NOT EXISTS holidays (
    id TEXT PRIMARY KEY,
    date TEXT NOT NULL UNIQUE,      -- YYYY-MM-DD形式
    name TEXT NOT NULL,              -- 祝日名
    type TEXT NOT NULL,              -- 'national', 'observance', 'company'等
    is_workday INTEGER DEFAULT 0,    -- 振替休日など特別対応用
    created_at TEXT DEFAULT (datetime('now')),
    updated_at TEXT DEFAULT (datetime('now'))
);
```

- 2025年・2026年の祝日データを初期データとして投入
- インデックスを設定（date, type）

### 3. バックエンドAPI実装
`functions/api/holidays/` ディレクトリを作成：

#### `index.ts`
- `GET /api/holidays` - 祝日一覧取得（年指定可能）
- `POST /api/holidays` - 祝日登録（手動追加用）

#### `[id].ts`
- `PUT /api/holidays/:id` - 祝日更新
- `DELETE /api/holidays/:id` - 祝日削除

#### `sync.ts`
- `GET /api/holidays/sync` - @holiday-jp/holiday_jpパッケージと同期

### 4. フロントエンドAPI追加
`src/lib/api.ts` に以下を追加：

```typescript
export const getHolidays = async (year?: number): Promise<Holiday[]>
export const createHoliday = async (holiday: Omit<Holiday, 'id' | 'created_at' | 'updated_at'>): Promise<string>
export const updateHoliday = async (id: string, data: Partial<Holiday>): Promise<void>
export const deleteHoliday = async (id: string): Promise<void>
export const syncHolidays = async (year?: number): Promise<{ success: boolean; message: string; synced: number; skipped: number }>
```

### 5. 型定義追加
`src/types/index.ts` に追加：

```typescript
export interface Holiday {
    id: string;
    date: string;           // YYYY-MM-DD
    name: string;           // 祝日名
    type: 'national' | 'observance' | 'company';
    isWorkday: boolean;     // 振替休日等の特別対応用
    created_at?: string;
    updated_at?: string;
}
```

### 6. 祝日表示（カレンダー）
`src/features/schedule/SchedulePage.tsx` を修正：

- 祝日データを取得して状態として保持
- `dateCellWrapper` コンポーネントをカスタマイズ
- 祝日の日付に赤い背景色と祝日名を表示
- シフト自動生成時に祝日データを渡す

### 7. 祝日ユーティリティ
`src/lib/holidayUtils.ts` を作成：

```typescript
export const createHolidayMap = (holidays: Holiday[]): Map<string, Holiday>
export const isHoliday = (dateStr: string, holidayMap: Map<string, Holiday>): boolean
export const getHolidayInfo = (dateStr: string, holidayMap: Map<string, Holiday>): Holiday | undefined
export const isWeekendOrHoliday = (dateStr: string, holidayMap: Map<string, Holiday>): boolean
export const getHolidayName = (dateStr: string, holidayMap: Map<string, Holiday>): string
```
