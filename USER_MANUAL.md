## 1. システム概要

### 1.1 システムの構成

トップページ（`/`）にアクセスすると、2つの入口が表示されます。

| 入口 | 対象 | 説明 |
|------|------|------|
| **スタッフ用ポータル** | 一般スタッフ | 希望休の提出・シフトの確認 |
| **管理者用ポータル** | シフト管理者 | スタッフ管理・シフト作成・各種設定 |

### 1.2 v2.0 の主な新機能

- スタッフが自分でログインして希望休を申請できるポータルを追加
- 希望休で「終日休」と「時間帯指定（一部不可）」を選択可能に
- 管理画面から営業時間・休館日を設定可能に
- シフト確定（ロック）機能の追加
- シフトのタイムライン表示（バー表示）をスタッフページにも追加
- 役職名を「スタッフ区分」に統一

### 1.3 v2.1 の変更点

- **アクセスキーを6桁に変更**: スタッフログインに使用するアクセスキーが4桁から6桁になりました（既存のキーは管理者が再設定してください）
- **Excel出力の改善**: ファイル名が `シフト表_YYYY-MM.xlsx` 形式に統一され、先頭に年月のタイトル行が追加されました。また、休日スタッフの表示が希望休・固定休・研修で色分けされ、定休日のスタッフは除外されるようになりました
- **Excel出力設定の追加**: 管理画面に「Excel出力設定」メニューが追加されました。土曜日の休日スタッフ非表示や、スタッフ別のシフトハイライトルールを設定できます
- **PDF出力を一時停止**: PDF出力ボタンは現在無効化されています
- **モバイル対応の改善**: 月間カレンダーのスクロール対応、スマホでの希望休管理画面のスタッフ選択をドロップダウン形式に変更
- **スタッフ・時間パターンの初期値改善**: 新規作成時に営業日設定に基づいた曜日が自動選択されるようになりました

## 2. スタッフ向けの使い方

### 2.1 スタッフ用ログイン

1. ブラウザでシステムのURL（例: `https://[システムドメイン]/`）にアクセスします。
2. 「スタッフ用ポータル」ボタンをクリックします。
3. ログイン画面が表示されます。
   - **名前**: ドロップダウンから自分の名前を選択します。
   - **アクセスキー**: 管理者から案内された数字のキーを入力します。
4. 「ログイン」ボタンをクリックします。

> **ログインできない場合**: 名前の選択ミス、またはアクセスキーの入力ミスが考えられます。管理者に確認してください。

---

### 2.2 希望休の提出

ログイン後、「希望休」タブが表示されます。毎月の提出期限までに希望を入力してください。

#### カレンダーの見方

| 表示 | 意味 |
|------|------|
| 白背景 | 出勤可能（未入力） |
| 赤背景「休」 | 終日休み希望 |
| 赤背景「部分休」 | 一部時間の休み希望 |
| 灰色（操作不可） | 固定休日（毎週の休み）または研修日 |
| 紺色の点 | すでにシフトが割り当てられている日 |

#### 希望を入力する

1. カレンダー上の日付をクリックすると、選択メニューが表示されます。
2. 以下から選択します。

**終日お休み**
- その日を丸1日休みとして申請します。

**一部の時間だけ不可**
- 出勤できない時間帯（開始時刻〜終了時刻）を指定します。
- 例: 午前中は用事があるため「09:00〜12:00 は不可」と指定する。

**就業可能（クリア）**
- すでに入力した希望休を取り消します。

#### 保存する

- 入力が終わったら「希望を保存」ボタンをクリックします。
- 提出期限前であれば何度でも修正できます。
- 「キャンセル」をクリックすると、未保存の変更を破棄できます。

---

### 2.3 シフトの確認

「シフト確認」タブでは、確定済みのシフトをタイムライン形式で確認できます。

- **自分のシフト**: ハイライト表示されます。
- **他のスタッフのシフト**: 同じ日の全員のシフトが表示されます。
- 月の切り替えは画面上部の矢印ボタンで行います。

---

### 2.4 ログアウト

「設定」タブを開き、「ログアウト」ボタンをクリックします。

---

## 3. 管理者向けの使い方

### 3.1 管理者用ログイン

1. トップページで「管理者用ポータル」ボタンをクリックします。
2. メールアドレスとパスワードを入力してログインします。

ログイン後、左側のメニューから各機能にアクセスできます。

---

### 3.2 初期設定

シフト作成を始める前に、設定メニューから事業所のルールを登録してください。

#### 時間パターン設定（`設定 > 時間パターン`）

「早番」「遅番」などの勤務時間帯を登録します。シフト作成時やスタッフの希望入力時に選択肢として使用されます。

| 項目 | 説明 |
|------|------|
| パターン名 | 例: 「早番」「遅番」 |
| 開始・終了時刻 | 例: 09:00 〜 18:00 |
| 対応曜日 | 利用可能な曜日にチェック |
| 対象スタッフ区分 | このパターンを使用できるスタッフ区分を選択 |

並び順はドラッグ&ドロップで変更できます。

#### スタッフ区分設定（`設定 > スタッフ区分`）

「正社員」「パート」などのスタッフ区分を登録します。

| 項目 | 説明 |
|------|------|
| 区分名 | 例: 「正社員」「アルバイト」 |
| 月間目標時間 | この区分の月間の目標勤務時間 |
| 週間目標時間 | この区分の週間の目標勤務時間 |
| 利用可能パターン | 使用できる時間パターンにチェック |

#### クラス設定（`設定 > クラス`）

業務の種類や担当グループを登録します。シフト表の表示分類に使用されます。

| 項目 | 説明 |
|------|------|
| クラス名 | 例: 「受付」「講師」 |
| 表示色 | タイムラインバーの色 |
| 自動配置 | 自動生成時に対象とするか |

#### 必要人数設定（`設定 > 必要人数`）

曜日・時間帯ごとの最低配置人数を登録します。シフト自動生成の基準として使用されます。

| 項目 | 説明 |
|------|------|
| 曜日 | 毎日 / 平日 / 特定曜日から選択 |
| クラス | 対象のクラス |
| 開始・終了時刻 | 必要人数が適用される時間帯 |
| 最低人数 | この時間帯に必要なスタッフ数 |
| 優先度 | 高いほど自動生成時に優先される（1〜5） |

並び順はドラッグ&ドロップで変更できます。

---

### 3.3 スタッフ管理

左メニューの「スタッフ」から、スタッフ情報を管理します。

#### スタッフの登録・編集

「新規追加」ボタンまたは各スタッフ行の編集アイコンをクリックします。

| 項目 | 説明 |
|------|------|
| 名前 | スタッフの氏名 |
| スタッフ区分 | 正社員・パートなどの区分 |
| 月間目標時間 | 個別に設定する場合（未入力の場合は区分のデフォルト値） |
| 週間目標時間 | 個別に設定する場合 |
| 所属クラス | 担当する業務クラス（複数選択可） |
| アクセスキー | スタッフがログインに使用する6桁の数字キー |

#### スタッフの並び替え

スタッフ一覧の各行の左端にあるアイコン（⠿）をドラッグして並び替えます。シフト表の表示順に反映されます。

#### スタッフの削除

編集画面またはスタッフ行の削除アイコンから削除できます。確認ダイアログが表示されます。

---

### 3.4 希望休の管理（管理者用）

左メニューの「希望休」から、スタッフが提出した希望休を確認・修正できます。

1. 画面上部でスタッフを選択します。
2. 対象月を選択します（前月・翌月への切り替えボタン使用）。
3. カレンダーに選択したスタッフの希望休が表示されます。
4. 日付をクリックして、希望内容の確認・修正ができます。

---

### 3.5 シフトの作成と調整

左メニューの「スケジュール」からシフト管理を行います。

#### ビューの切り替え

画面上部のタブで表示形式を切り替えられます。

| ビュー | 説明 |
|--------|------|
| 月ビュー | カレンダー形式で1ヶ月のシフトを表示 |
| 週ビュー | 週単位のタイムラインで表示 |
| 日ビュー | 1日のタイムラインで詳細な編集が可能 |

#### シフトの自動生成

1. 画面右上の「生成」ボタンをクリックします。
2. 確認ダイアログが表示されます（既存のシフトは上書きされます）。
3. 「実行」をクリックすると、以下の条件を考慮してシフトが自動生成されます。
   - スタッフが提出した希望休
   - 各スタッフのスタッフ区分と利用可能な時間パターン
   - 必要人数設定のルール
   - ロック済みの日付（後述）のシフトは保持されます

#### シフトの手動編集（日ビュー）

日ビューでは以下の操作が可能です。

**シフトの追加**
- 「追加」ボタンをクリックし、スタッフを選択します。
- 日付・開始時刻・終了時刻を設定して保存します。

**シフトの入れ替え**
- 既存のシフトバーをクリックし、「入れ替え」を選択します。
- 他のスタッフを選択すると入れ替えが実行されます。

**シフト時間の調整**
- タイムライン上のバーをドラッグして開始・終了時刻を変更できます（15分単位）。
  - バーをドラッグ: 開始時刻の変更
  - Alt + ドラッグ: 終了時刻の変更

**シフトの削除**
- シフトバーをクリックし、削除アイコンをクリックします。確認後に削除されます。

#### シフトのロック（確定）

特定の日付のシフトを確定（ロック）すると、次回の自動生成時に上書きされなくなります。

- 日ビューの日付ヘッダーにあるロックアイコンをクリックしてロック/解除を切り替えます。
- ロック済みの日付はアイコンが変わり、自動生成から除外されます。
- ロック後も管理者による手動編集は可能です。

#### シフトの一括クリア

「クリア」ボタンをクリックすると、表示中の月のシフトを全削除できます（ロック済みの日付は除外されます）。

---

### 3.6 シフトのエクスポート

確定済みのシフトをファイルとして出力できます。

| 形式 | 説明 |
|------|------|
| **Excel** | タイムライン付きの詳細なシフト表（.xlsx 形式）。ファイル名は `シフト表_YYYY-MM.xlsx` 形式 |
| **PDF** | 現在は一時停止中（ボタン無効） |

画面右上の「Excel」ボタンからダウンロードできます。

Excel出力には以下の情報が含まれます。

- 先頭行に年月のタイトル
- 出勤スタッフのシフトバー（時間ごとの細線付き）
- 休日スタッフの情報（希望休・固定休・研修を色分けで表示）
- 定休日のスタッフは行から除外

> Excel出力のカスタマイズは「**設定 > Excel出力設定**」から行えます（次節参照）。

---

### 3.7 Excel出力設定

`設定 > Excel出力設定` から、Excel出力の見た目をカスタマイズできます。

#### 土曜日の表示制御

| 設定 | 説明 |
|------|------|
| **土曜日の休日スタッフを非表示にする** | ONにすると、土曜日の行には出勤スタッフのみが表示されます（休日のスタッフ行は省略） |

#### スタッフ別シフトハイライト

特定のスタッフが「通常の勤務時間」と異なるシフトに入った際、Excel上でその行を指定した色で強調表示するルールを設定できます。

**ルールの追加手順**

1. 「ルールを追加」ボタンをクリックします。
2. 対象スタッフ・通常の勤務時間（開始〜終了）・ハイライト色を設定します。
3. 「設定を保存」ボタンをクリックします。

> **ルールの動作**: 設定した「通常の勤務時間」と **一致しない** シフトが組まれた場合のみ、その行が指定色で塗りつぶされます。通常通りのシフトには色がつきません。

## 4. トラブルシューティング

**Q. スタッフ用ログインで「認証失敗」と表示される**
> A. 名前の選択ミス、またはアクセスキーの入力ミスが考えられます。管理者に正しいアクセスキーを確認してください。

**Q. 希望休の保存ボタンが押せない**
> A. 変更が検出されていない可能性があります。日付をクリックして希望内容を変更してから再度お試しください。

**Q. 自動生成したシフトに空白（割り当てなし）が多い**
> A. 「必要人数設定」を満たす出勤希望が集まっていないか、対応できるスタッフが不足しています。スタッフの希望休を確認するか、手動でシフトを追加してください。

**Q. 特定の日だけシフトが自動生成されない**
> A. その日がロック済みになっている可能性があります。日ビューでロックアイコンを確認し、解除してから再生成してください。

**Q. スタッフのシフトバーがオレンジ色で表示される**
> A. そのスタッフが希望休を提出しているにもかかわらずシフトに入っている日です。管理者が内容を確認・調整してください。
